# TaskFlow Dashboard

TaskFlow is a scalable task management web application built with React and FastAPI.

## Features
- JWT Authentication
- Protected Routes
- Task CRUD Operations
- Dashboard Analytics
- Responsive UI
- Secure API Integration

## Tech Stack
Frontend:
- React + TypeScript
- Tailwind CSS
- React Query

Backend:
- FastAPI
- PostgreSQL / SQLite
- JWT Authentication

## Setup Instructions
1. Run backend
2. Run frontend
